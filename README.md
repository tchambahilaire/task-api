# Task API - Documentation Complète

## 📝 Description

API REST complète de gestion de tâches construite avec **Express**, **TypeScript**, **Zod** et **Winston**. 
Cette API démontre l'utilisation de bonnes pratiques de développement, une architecture propre et une documentation exhaustive.

## ✨ Fonctionnalités

- ✅ **CRUD complet** des tâches
- ✅ **Pagination** avancée
- ✅ **Filtrage** par statut et recherche textuelle
- ✅ **Tri** par date et titre
- ✅ **Validation** des données avec Zod
- ✅ **Middleware d'erreur** global
- ✅ **Rate limiting** pour la sécurité
- ✅ **Logging** avec Winston (console + fichiers)
- ✅ **Documentation Swagger** interactive
- ✅ **Tests unitaires** avec Jest
- ✅ **CORS** configuré
- ✅ **Compression** des réponses
- ✅ **Helmet** pour la sécurité
- ✅ **Variables d'environnement**
- ✅ **Stockage en mémoire** (Map)
- ✅ **Déploiement** sur Render/Railway

## 🚀 Installation

```bash
# Cloner le projet
git clone https://github.com/ton-username/task-api.git

# Installer les dépendances
cd task-api
npm install

# Copier le fichier d'environnement
cp .env.example .env

# Démarrer en mode développement
npm run dev
